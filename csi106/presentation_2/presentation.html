asymmetric cryptography

Cryptography involving keypairs

The keys form a pair

Private key: secret, must not be shared

Public key: can be shared

Public key encrypt -> Private key decrypt

Private key encrypt -> Public key decrypt

Consequence

No secret has to be shared to encrypt/decrypt messages securely.
